## 렌더링 엔진

요청한 컨텐츠들을 화면에 출력해주는 소프트웨어이다. 이 소프트웨어는 렌더링 과정에서 해당 과정을 거친다.

1. HTML 파싱 후, DOM 트리 구축
2. CSS 파싱 후, CSSDOM 트리 구축
3. JavaScript 시행
4. DOM 트리와 CSSDOM을 조합하여 Render트리 구축 
5. 각 노드의 position을 계산 후, 화면에 그려줌!
